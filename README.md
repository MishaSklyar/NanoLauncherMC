The **Simplest** and **Most Powerful** Minecraft Launcher for macOS, Windows, and Linux.
Built with Python and Dear PyGui. No Electron. No Bloat. Blatantly Fast.

**Repository**: [MishaSklyar/NanoLauncherMC](https://github.com/MishaSklyar/NanoLauncherMC)

## Features
*   **Blazing Fast**: Optimized multi-threaded downloading.
*   **Lightweight**: Uses Dear PyGui (GPU accelerated IM-GUI) - runs on anything.
*   **Mod Support**: One-click install for Fabric, Forge, and Quilt.
*   **Modrinth Integration**: Search and install mods directly from the launcher.
*   **Optimization**: Auto-injects Aikar's Flags for maximum FPS.
*   **Portable**: Self-contained installation.

## Installation

1.  Clone the repo:
    ```bash
    git clone https://github.com/MishaSklyar/NanoLauncherMC.git
    cd NanoLauncherMC
    ```
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Run:
    ```bash
    python3 run.py
    ```

## Requirements
